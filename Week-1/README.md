# Week 1 - Red Teaming Lab Report

## Overview
This week focused on practical red teaming techniques in a controlled lab using Metasploitable 2 as the target and Kali Linux as the attacker machine.

## Tasks Completed

- Network reconnaissance with Nmap
- Vulnerability assessment with OpenVAS
- Exploitation using Metasploit Framework
- Root shell access via multiple attack paths
- Password cracking using John the Ripper
- Online brute-force testing with Hydra
- Malware test file analysis using EICAR
- MITRE ATT&CK mapping of activities

## Tools Used

- Kali Linux
- Nmap
- Metasploit
- OpenVAS
- Hydra
- John the Ripper
- Netcat
- VirusTotal
- Hybrid Analysis
